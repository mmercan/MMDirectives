restrict: 'A'

wrapper around http://digitalbush.com/projects/masked-input-plugin/ (file included)

NameTypeDesciption
mmMask string mmMask define the mask
'9': [0-9], 'a': [A-Za-z], '*': [A-Za-z0-9]
like : mm-mask="a*-999-a999"
