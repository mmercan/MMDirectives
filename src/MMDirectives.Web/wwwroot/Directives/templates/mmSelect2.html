mmSelect2

restrict: 'A'

very easy to use
1- create your select normally 
<select class="form-control" ng-model="selectedListItem">
  <optgroup ng-repeat=" group in timezonelist" label="{{group.label}}">
    <option ng-repeat="item in group.cities" value="{{item.value}}">{{item.name}}</option>
  </optgroup>
</select>

2- test without Select2
3- mm-select2="{ allowClear: true}"> 
<select class="form-control" ng-model="selectedListItem" mm-select2="{ allowClear: true}">
  <optgroup ng-repeat=" group in timezonelist" label="{{group.label}}">
    <option ng-repeat="item in group.cities" value="{{item.value}}">{{item.name}}</option>
  </optgroup>
</select>
if you want to be free on options use input instead adding tag to your mm-select2 is required (for now) 
 <input mm-select2="{  tags: []}" class="form-control" name="to" id="mail-to" ng-model="mailto">
if you want to provide pre-defined flexible options use mm-select2 tag array for variables 
 <input mm-select2="{  tags: [{'id':'1','text':'1'},{'id':'2','text':'2'},]}" class="form-control" name="to" id="mail-to" ng-model="mailto">
